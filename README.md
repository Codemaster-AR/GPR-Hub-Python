# GPR_Reader_Identify_Python_Beta
A python3 based CLI that utilizes the power of AI to answer user queries and help analyze GPR images. This is an experimental version and I am not responsible for any misuse of this software.

**Parts:**
  - MacOS (Sequoia and later)
  - Windows (11)
  - Linux
**Chrome is recommened for this installation - on any operating system**
**MacOS installation instructions:**
  Downloading python3 (if not done already):
    1) 	Visit https://www.python.org/downloads/ and download python 3.14.0 (latest for MacOS as of November 2025). You should see a file named 'python-3.14.0-macos11.pkg' or
    		something like that appear in your downloads folder.
    2) 	Double click the installer file (.pkg) and follow the instructions that will display on your screen. Accept the default settings, it works well for majortiy of the
    		users. You may be prompted to enter your password.
		3)  After the installation is completed, you should open/restart the app 'terminal' and run the following commands:
  			- 'python3 --version'
